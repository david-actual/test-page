# Test Page

This project is a simple static landing page created as part of a CI/CD practice using GitHub, Azure DevOps, Docker, and Azure Container Registry (ACR). The application consists of a single `index.html` file packaged into a Docker image and served through **Nginx**.

## Why Nginx?

The project uses the official **Nginx** Docker image because it is lightweight, easy to configure, and designed to serve static web content. While a Node.js-based solution using `http-server` would also work, Nginx provides a simpler approach since no server-side code or additional dependencies are required.
